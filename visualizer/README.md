# MIPT-V Module topology visualizer tool
## Requirements
* npm >= 6
* node >= 8

[Install npm and node.js](https://www.npmjs.com/get-npm)

## Installing
### Build
    npm install
### Run
    npm run start -- --env.path=/path/to/topology.json
    Project runs at http::/localhost:8080/
### Tests
    npm run test
